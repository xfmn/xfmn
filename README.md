视频缓冲放大电路（高频）
三极管型号：2N2222A、2SC3356、BF199、2N918 等高频低电容三极管

视频信号频率较高（NTSC约6MHz，PAL约5MHz），推荐共集（射极跟随）电路
视频信号 ───┬──R1──┬──B
            │       │
          C1      Q(NPN)
            │       │
           GND     E─── 输出（视频OUT）
                   │
                  R2
                   │
                  GND
C1：输入耦合电容（~10nF）
R1：偏置电阻（~1k）
R2：发射极电阻（~75Ω 匹配视频阻抗）
共集结构保持电压但提高电流输出能
力

📶 信号带宽	视频信号要求较高频带宽（>6MHz），低频三极管会失真模糊
⚡ 阻抗匹配	视频输出阻抗一般为 75Ω，电路需加匹配电阻
🔊 音频隔直	电容耦合输入输出，避免直流进入音频设备
